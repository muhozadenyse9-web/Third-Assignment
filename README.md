# Campus Information Website

## Project Description
This project is a single-page campus information website that shows services, locations, announcements, and layout behaviors using HTML and CSS.

## Layout Features Demonstrated
- Navigation bar with dropdown menu
- Card layout for services
- Multi-column gallery layout
- Sticky announcement notice
- Display behaviors (block, inline, none)
